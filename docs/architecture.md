# CallWise-AI 系统架构

## 整体架构

```
[销售对话] → [手机APP] → [Node.js API] → [AI分析] → [辅导反馈]
                                    ↓
                              [SQLite + 云端]
```

## 核心组件

### 1. 移动端 (React Native)
- 语音录制APP
- 离线存储
- 实时同步

### 2. 后端 (Node.js)
- REST API
- AI分析服务
- 数据同步

### 3. 数据库 (SQLite + 云端)
- 对话记录
- 分析结果
- 用户配置

### 4. AI服务
- 中文语音识别
- 情感分析
- 行为评分

## 数据流

1. 销售使用手机APP录制对话
2. 本地存储并同步到云端
3. AI服务分析对话内容
4. 生成评分和辅导建议
5. 销售经理查看反馈

## 部署

- 单服务器部署
- Docker容器化
- 环境变量配置
